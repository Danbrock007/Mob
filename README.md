# PTI Premium POS v1.3

Android WooCommerce POS for Pet Trade Innovations.

## Included
- Official PTI logo and supplied application icon downloaded during GitHub build
- Dashboard analytics and top three recent customers
- Live order search by customer name, email, phone, order ID and order number
- Reliable pagination for orders, products and POS products
- Premium invoice PDF/image with customer contact, address, products, totals, status and payment gateway
- Order attribution shown in app details but excluded from invoice
- Add/edit products including images and gallery
- Three-step POS: products → customer → review/payment → complete order
- Remembered login with separate logout/forget behavior

Install the connector plugin from `wordpress-plugin/pet-trade-innovations-pos` before using the app.
